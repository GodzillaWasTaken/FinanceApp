## Installation (self hosted)

To install and set up the project instantly, run:

```bash
curl -sSL [https://raw.githubusercontent.com/GodzillaWasTaken/FinanceApp/main/install.sh](https://raw.githubusercontent.com/GodzillaWasTaken/FinanceApp/main/install.sh) | bash
```

it will download a `docker-compose.yml` and a `.env_example` file that you need to rename and fill in with your database credentials.
